# Nerva

## Description

Nerva is a board game that utilizes a standard chess board and backgammon pieces.

3 8x8 boards side by side
Left Origin Right

turn based

white vs black

press on an empty tile to capture it

Each tile:
has one attack point
attacks on every diagonal

A tile on all 3 board in same location is 3 points

Attack points only transfer through the 3 boards or diagonally on the same board

Only 3 tiles of the same color can be place next to each other horizontally or diagonally.
